# 使用教程

这篇文档从**用户视角**讲整套流程怎么走、每一步会遇到什么、出问题怎么办。
关于技能内部怎么实现的，看 [SKILL.md](../pta-auto-submit/SKILL.md) 与 [references](../pta-auto-submit/references/)。

---

## 一句话触发

启动 Claude Code 后，直接说（任一即可）：

```
帮我把这套 PTA 题做了
帮我刷这套 PTA
PTA 自动答题
帮我把 PTA 上这套编程题写了
```

Claude 加载技能 → 准备 Playwright MCP → 打开浏览器 → 进入 PTA 登录页。

## 第一步：登录

技能在 `https://pintia.cn/home` 停下，请你**自己输入账号密码**。

> ⚠️ 这是**必须由你亲自操作**的一步。技能绝不猜测、存储、回显你的凭据。

## 第二步：选择题目范围

进入后，技能会抓取平台展示的所有题目分类（如"第 3 套 - 循环结构"），并停下请你选。

这是第二个**必须确认**的点——防止误抓大量无关题目。

### 示例对话

```
Claude: 平台共 6 套题：
  1. 第 1 套 - 顺序结构（10 题）
  2. 第 2 套 - 分支结构（15 题）
  3. 第 3 套 - 循环结构（20 题）
  ...
  你想做哪一套？回复编号即可。

你: 3

Claude: 收到，「第 3 套 - 循环结构」共 20 题，开始抓取...
       [翻页中 ... 1/20 ... 5/20 ...]
```

你只需回一个编号；翻页和抓题全自动，不必盯着。

---

## 第三步：连续解题与提交

确定范围后，技能进入连续循环——**默认不会再每道题都问**。

### 各类题型怎么处理

| 题型 | 动作 |
|---|---|
| 判断题 | 点选 ✓ 或 ✗ |
| 单选题 | 点选正确选项 |
| 多选题 | 点选所有应选项，再保存 |
| 编程题 | 生成完整可运行代码（含 `main`、`#include`）后提交 |
| 函数题 | 仅实现函数体，**不带** `main` / `#include`（裁判自动提供上下文） |
| 带图题 | VLM 识别 → 解答；识别失败停下请文字描述 |

### 单题自愈过程（活例子）

假设一道编程题提交后判"答案错误"：

```
第 7 题（25 分）：输出结果与预期不符
  → 实际：[1, 2, 3, 4, 5]
  → 预期：[5, 4, 3, 2, 1]
  → 原因推测：排序方向相反
  → 第 1 次重试（改降序）... 提交... 评测中...
  → 评测通过 ✓
```

如果错误较复杂（编译错误 / 段错误 / 超时），技能会把编译器输出 + 错误类型一起交给模型重新生成，最多自愈 **5 次**。超过则如实标记该题未 AC，跳到下一题，**不会空转**。

### 评测结果解读

| 结果 | 含义 |
|---|---|
| 答案正确 | 完全 AC |
| 部分正确 | 部分用例通过（多见于编程题，按 AC 率得分）|
| 答案错误 | 输出不匹配 |
| 编译错误 | 代码 gcc/g++ 没通过 |
| 运行超时 | 时间超限，算法需要优化 |
| 段错误 | 越界访问，数组/指针常见 |
| 内存超限 | 分配过大，通常是动态内存失控 |
| 格式错误 | 多余/缺少空格换行，注意输出格式 |

## 批量做题的上下文提醒

刷 20+ 题会让 Claude Code 上下文变长。技能会在合适时机主动提醒：

> 上下文已经比较长了，建议你输入 `/clear` 开新会话，技能会配合断点续做继续执行。

**别担心**——断点续做机制会让你 `/clear` 后再进页面时，自动恢复到未完成的题。

---

## 断点续做

跳题、`/clear`、关了 Claude 再打开——都会触发页面重新加载。

技能用 `sessionStorage` 记状态：

| 状态 | 恢复行为 |
|---|---|
| `running` | 直接跳到下一道未 AC 的题 |
| `solver-navigation` | 正在跳题途中，继续完成本次导航 |
| 无标记 | 视同新任务，重新确认题目范围 |

恢复后流程无缝继续，不必重头选范围、重做已 AC 的题。

---

## 中途想停下

- 想暂停：直接关掉 Claude，下次打开进 PTA 页面会自动续做
- 想终止：清掉 sessionStorage（按 F12 → Application → Storage → 删 `pta-auto-submit` 开头的 key）
- 想从头来：同上清掉标记，重新触发一句"帮我把 PTA 题做了"

---

## 典型耗时参考

| 规模 | 大致耗时 |
|---|---|
| 10 题（全选择/判断） | 2-5 分钟 |
| 20 题（编程 + 选择混合） | 10-25 分钟 |
| 全 AC + 批量提交 | 看评测排队时间，可能额外 1-3 分钟 |
